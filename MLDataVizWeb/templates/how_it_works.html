How Website is Working

- It is an application that shows basketball players' balance scores via their performance stats.

- You can register our Wii Board Analyzer Application with a username and a password.

- If you already have an account, you can log in directly.

- After you log in to the application, you will see 4 exercises on the main page.

- When you click on these exercises, you can see all our basketball players registered in our database.

- With the search bar in our application, you can easily filter the player you want, to avoid confusion on the main page.

- When you click on any of these basketball players, you can examine the basketball players' data with various graphs.

How System is Working

Our system predicts basketball players' balance scores using machine learning algorithms. For each exercise, we estimate the balance score using copx and copy values from the 10-column performance statistics of basketball players. Our model is trained with the Random Forest algorithm. Model training was carried out by properly separating the given basketball players' data into train and test sets. Our product also includes data analysis graphs that you can access for 5 basketball players.
